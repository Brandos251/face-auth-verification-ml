Backend – Сервис верификации лиц с использованием ML

## Описание

Этот бэкенд-сервис предоставляет API для проверки изображений лиц с помощью
глубокой нейронной сети на основе **MobileNetV2**.

---

## Функциональность

- Предобработка изображений
- Прогноз подлинности лица
- REST API для взаимодействия с фронтендом
- Пайплайн обучения модели

---

## Описание файлов

- `training.py` — скрипт для обучения модели
- `preprocessing.py` — загрузка и предобработка датасета
- `server.py` — FastAPI сервер с endpoint для предсказаний
- `main.py` — тест сервиса / endpoint для проверки статуса
- `requirements.txt` — зависимости проекта

---

## Запуск сервера

```bash
pip install -r requirements.txt
uvicorn server:app --reload
