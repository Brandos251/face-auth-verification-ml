## Описание

Этот бэкенд-сервис предоставляет API для проверки изображений лиц с помощью
глубокой нейронной сети на основе **MobileNetV2**.

---

## Основные функции

- предобработка входных изображений
- классификация изображений лиц (Real / Fake)
- REST API для взаимодействия с мобильным приложением
- обучение и сохранение модели машинного обучения
- загрузка обученной модели для инференса

---

## Используемые технологии

- Python
- FastAPI
- TensorFlow / Keras
- MobileNetV2
- NumPy
- Pillow
- Uvicorn

---

## Структура и назначение файлов

- `training.py`  
  Скрипт обучения модели машинного обучения.
  Отвечает за:
  - создание архитектуры MobileNetV2
  - обучение модели
  - сохранение контрольных точек и финальной модели (`mobilenet_final.h5`)
  - оценку качества на тестовой выборке

- `preprocessing.py`  
  Модуль загрузки и предобработки данных.
  Реализует:
  - чтение изображений из датасета
  - изменение размера изображений
  - нормализацию пикселей
  - формирование `tf.data.Dataset` для обучения, валидации и тестирования

- `server.py`  
  Основной серверный файл FastAPI.
  Отвечает за:
  - загрузку обученной модели
  - приём изображений от клиента
  - выполнение инференса
  - возврат результата в формате JSON

- `main.py`  
  Вспомогательный файл для проверки работоспособности сервиса.
  Возвращает информацию о доступности FastAPI и версиях используемых библиотек

- `requirements.txt`  
  Список зависимостей, необходимых для запуска сервиса

## Запуск сервера

1. Установить зависимости:
```bash
pip install -r requirements.txt
```
2. Запустить сервер:
```bash
uvicorn server:app --reload
```
3. После запуска сервер будет доступен по адресу:
```text
http://localhost:8000
```
## API
### POST `/predict`
Принимает изображение лица и возвращает результат верификации

Пример ответа:
```json
{
  "result": "Изображение подлинное на 87%"
}
```
