## Описание

Данное Android-приложение является клиентской частью системы
верификации изображений лиц на основе методов машинного обучения.

Приложение позволяет пользователю выбрать изображение лица с устройства,
отправить его на серверную часть и получить результат проверки подлинности
в виде процентной оценки

---

## Назначение приложения

Frontend-часть разработана для:
- взаимодействия пользователя с системой верификации
- отправки изображений лиц на сервер
- отображения результата работы модели машинного обучения

Приложение служит пользовательским интерфейсом для демонстрации
работы алгоритмов верификации изображений лиц

---

## Основная функциональность

- выбор изображения лица из памяти устройства
- отправка изображения на сервер по HTTP
- получение результата верификации от backend-сервиса
- отображение процента подлинности изображения пользователю

---

## Используемые технологии

- Kotlin
- Android SDK
- REST API
- HTTP-запросы к серверу FastAPI

---

## Принцип работы приложения

1. Пользователь запускает приложение
2. Выбирает изображение лица на устройстве
3. Приложение отправляет изображение на backend-сервер
4. Сервер обрабатывает изображение с помощью модели машинного обучения
5. Приложение получает и отображает результат верификации

---

## Взаимодействие с backend

Frontend взаимодействует с серверной частью через REST API

Для корректной работы приложения необходимо:
- запущенный backend-сервер
- корректно настроенный адрес сервера в приложении
