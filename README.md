# Система верификации изображений лиц (ML)

## Дипломный проект

Данный репозиторий содержит дипломную работу 
**«Система верификации изображений лиц на основе методов машинного обучения»**.

Проект представляет собой полноценную систему для проверки подлинности изображений лиц 
(реальные фотографии vs изображения, сгенерированные ИИ) с использованием методов глубокого обученияg.

---

## Описание проекта

Система анализирует входное изображение лица и определяет, является ли оно:
- **реальной фотографией**, или
- **синтетически сгенерированным (поддельным) изображением**

Решение основано на сверточной нейронной сети (MobileNetV2)
и возвращает процентную оценку уверенности в результате.

## Принцип работы системы

1. Пользователь загружает изображение лица через мобильное приложение
2. Изображение отправляется на серверную часть (FastAPI)
3. Модель машинного обучения обрабатывает изображение
4. Система возвращает процентную оценку подлинности изображения
